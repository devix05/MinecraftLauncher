## ✨ Features

### 🎮 Game Management
- **Launch Minecraft** with any version (1.0 - latest)
- **Microsoft Account Login** with full OAuth 2.0 support
- **Offline Mode** for playing without an account
- **RAM Allocation** slider (1-16 GB)
- **JVM Arguments** customization
- **Fullscreen** toggle option
- **Auto-close** launcher after game start

### 🎨 Skin Studio
- **PNG Skin Selection** with live preview
- **3D-style** skin visualization
- **Steve/Alex** model selection
- **Upload skins** directly to Mojang servers
- **Saved skins library** with quick load

### 📦 Version Manager
- **List all installed** Minecraft versions
- **Download and install** any version automatically
- **Repair corrupted** versions
- **Search** for specific versions
- **Progress tracking** during installation

### 🎨 Modpacks & Mods (50+ included)
- **Utility Mods**: Simple Voice Chat, JEI, Xaero's Minimap, MiniHUD, Waila, Inventory Profiles, Replay Mod, WorldEdit, Litematica
- **Performance Mods**: Sodium, Iris Shaders, Lithium, Starlight, FerriteCore
- **Quality of Life**: Mouse Tweaks, AppleSkin, BetterF3, Chat Heads, Zoomify
- **Visual Mods**: Complementary Shaders, Fresh Animations, Falling Leaves
- **Complete Modpacks**: Better MC, RLCraft, SkyFactory, All The Mods, Pixelmon, Fabulously Optimized, and many more!

### 📁 Instance Management
- **Create multiple instances** with different settings
- **Import existing instances** from folders
- **Customize each instance** independently
- **Quick launch** from instance list

### ⚙️ Settings
- **Theme selection**: Dark / Light / System
- **Language**: German / English (fully translated)
- **Java path** customization
- **Minecraft directory** selection
- **Custom JVM arguments**
- **Logging** with save options

### 📊 Statistics
- **Playtime tracking** (hours/minutes)
- **Launch counter**
- **System information** display
- **Recent logs** viewer

## 🚀 Installation

### Prerequisites
- **Windows 10/11** (64-bit recommended)
- **Python 3.13** or higher
- **Java 8** or higher (for Minecraft)

### Step 1: Install Python
Download Python from [python.org](https://python.org) and make sure to check **"Add Python to PATH"** during installation.

### Step 2: Clone or Download
```bash
git clone https://github.com/yourusername/minecraft-modern-launcher.git
cd minecraft-modern-launcher
